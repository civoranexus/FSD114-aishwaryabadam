# EDUVillage

EDUVillage is a Full Stack web application built as part of the CivoraX Internship Program.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB

## Features
- User authentication
- Role-based access
- Course and content management
- API-based architecture

## Setup Instructions

### Frontend
```bash
cd frontend
npm install
npm run dev
